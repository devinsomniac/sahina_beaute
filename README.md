# Sahina Beauté 💅

> **Salon de Beauté Indien · Saint-Denis, France**  
> A women-only Indian beauty salon website — built with Next.js and deployed on Vercel.

🔗 **Live Demo:** [sahina-beaute.vercel.app](https://sahina-beaute.vercel.app)

---

## About the Project

Sahina Beauté is a professional beauty salon website for an Indian beauty salon based in Saint-Denis, France. The site is designed exclusively for women and offers a full range of beauty services including nails, waxing, facials, Ayurvedic massage, henna, lashes, and bridal packages.

The website is fully in French and serves as both an informational and booking platform for the salon's clientele.

---

## Features

- **Services Showcase** — Detailed descriptions and starting prices for all 8+ service categories
- **Full Price Menu** — Tabbed pricing list covering nails, waxing, facials, massage, henna, and more
- **Online Booking Form** — Appointment request form with service selection and date picker
- **Customer Reviews** — Client testimonial section with a review submission form
- **About Section** — Salon story, opening hours, and key highlights
- **Contact & Map** — Address, phone numbers, social links, and a Google Maps link
- **Opening Promo Banner** — 20% off all services for the opening period
- **Responsive Design** — Optimised for mobile and desktop
- **Walk-ins Welcome** — Clearly communicated throughout the UI

## Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/devinsomniac/sahina_beaute.git
cd sahina_beaute

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

The project is deployed on [Vercel](https://vercel.com). Any push to the `master` branch will trigger an automatic deployment.

To deploy your own instance:

1. Fork this repository
2. Import it into [Vercel](https://vercel.com/new)
3. Deploy — no additional configuration required
